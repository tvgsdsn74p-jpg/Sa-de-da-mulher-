# Saude-da-mulher-
# 🌸 Plataforma de Educação em Saúde da Mulher

## 📌 Sobre o projeto
Este projeto tem como objetivo desenvolver uma plataforma digital educativa voltada à saúde da mulher, com foco em orientação, prevenção e autocuidado.

## 🎯 ODS Relacionada
ODS 4 – Educação de Qualidade

## 🧠 Problema
Muitas mulheres não possuem acesso a informações confiáveis sobre saúde íntima, o que pode gerar dúvidas, medo e problemas de saúde.

## 🎯 Objetivo
Criar uma plataforma simples e acessível que ensine sobre:
- Higiene íntima
- Sintomas comuns
- Prevenção de doenças
- Quando procurar ajuda médica

## 📚 Conteúdos
- Corrimento: o que é normal
- Ardência e coceira
- Cuidados diários
- Mitos e verdades

## 💻 Tecnologias
- HTML
- GitHub

## 🌍 Impacto
Promover educação em saúde, reduzir a desinformação e ajudar mulheres no autocuidado.

## 👩‍⚕️ Público-alvo
Mulheres jovens e adultas, principalmente com pouco acesso à informação.
